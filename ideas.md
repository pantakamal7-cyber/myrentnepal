# MYRENT – Design Brainstorm

## Context
A hyper-local rental platform for Kathmandu, Nepal. Mobile-first, targeting users with basic smartphone literacy and slow mobile data. Core mission: eliminate brokers, prevent fake listings, enable direct landlord-to-tenant communication.

## Idea 1: "Nepali Terracotta & Ink" — Editorial Brutalism

**Design Movement:** South Asian Editorial Brutalism — inspired by Newari architecture's bold geometry, terracotta brickwork, and hand-stamped typography.

**Core Principles:**
1. Asymmetric, offset grid layouts that break the monotony of card-based rental sites
2. Heavy typographic hierarchy — oversized location labels as visual anchors
3. Warm earthy tones contrasted with sharp ink-black structural elements
4. Functional density — maximum information per screen without clutter

**Color Philosophy:** Terracotta (#C4622D) as the hero accent, paired with deep ink (#1A1208) for text, warm cream (#F5EFE0) as background, and a muted sage (#7A8C6E) for secondary badges. Emotional intent: trustworthy, grounded, local.

**Layout Paradigm:** Offset card grid with a left-anchored sidebar for filters. Hero section uses a full-bleed asymmetric split — 60% image, 40% search panel. No centered hero.

**Signature Elements:**
- Brick-pattern SVG texture overlay on hero section
- Bold ward/neighborhood labels in oversized display type
- "Verified" badge styled as a hand-stamp seal

**Interaction Philosophy:** Deliberate, confident interactions. Buttons have a slight press-down effect. Filter selections feel tactile.

**Animation:** Entrance animations at 180ms ease-out. Cards stagger in at 50ms intervals. No floating elements.

**Typography System:** Display: "Playfair Display" (bold, editorial). Body: "DM Sans" (clean, readable at small sizes).

## Idea 2: "Mountain Clarity" — Functional Minimalism with Altitude

**Design Movement:** Scandinavian Minimalism adapted for South Asian context — clean white space, sharp utility, with subtle Himalayan blue as the only accent.

**Core Principles:**
1. Radical whitespace as trust signal — sparse layouts signal legitimacy over cluttered broker sites
2. Data-first hierarchy — price and location are always the largest elements
3. Single accent color (deep Himalayan blue) used sparingly for CTAs only
4. Mobile-first grid that degrades gracefully on slow connections

**Color Philosophy:** Pure white (#FFFFFF) background, near-black (#111827) text, Himalayan blue (#1D4ED8) for primary actions, light slate (#F1F5F9) for card backgrounds. Intent: clinical trust, government-grade reliability.

**Layout Paradigm:** Full-width search bar as the page anchor. Below it, a horizontal filter strip. Listings in a 2-column mobile / 3-column desktop masonry grid. Left sidebar for desktop filters.

**Signature Elements:**
- Thin blue underline on verified listings
- Minimal icon set (phone, map pin, verified checkmark)
- Price displayed in large Devanagari-inspired numerals

**Interaction Philosophy:** Frictionless. Every action should require the fewest possible taps.

**Animation:** Subtle fade-in at 150ms. No decorative animations. Focus on speed.

**Typography System:** Display: "Space Grotesk" (geometric, modern). Body: "Noto Sans" (excellent Devanagari support).

## Idea 3: "Kathmandu Street" — Warm Urban Realism

**Design Movement:** Urban Vernacular — inspired by Kathmandu's street signage, tiled storefronts, and the warm amber glow of evening markets.

**Core Principles:**
1. Warm amber and saffron palette that feels local and human, not corporate
2. Photo-centric layouts — real property photos dominate, text is secondary
3. Neighborhood identity as a first-class design element
4. Bold, high-contrast CTAs that work in bright sunlight on mobile screens

**Color Philosophy:** Saffron amber (#F59E0B) as primary accent, deep charcoal (#1C1917) for text, warm off-white (#FAFAF9) background, with muted red (#DC2626) reserved exclusively for report/alert actions. Intent: warm, approachable, urgent where needed.

**Layout Paradigm:** Full-bleed hero with a floating search card. Listings use a Pinterest-style variable-height masonry grid. Neighborhood filter tabs scroll horizontally at the top.

**Signature Elements:**
- Neighborhood "district tags" styled as metro-map colored pills
- "Direct Call" button in high-contrast saffron with phone icon
- Report button in muted red, always visible but not dominant

**Interaction Philosophy:** Touch-optimized. Large tap targets (min 48px). Swipeable photo galleries on property cards.

**Animation:** Card hover lifts with shadow at 200ms. Photo galleries use swipe gestures. Price counter animates on filter change.

**Typography System:** Display: "Sora" (rounded, friendly). Body: "Inter" (pragmatic readability).

## Selected Design: **Idea 1 — "Nepali Terracotta & Ink"**

This approach best reflects the local identity of Kathmandu while maintaining the editorial authority needed to signal trustworthiness. The asymmetric layout differentiates MYRENT from generic rental sites, and the terracotta/ink palette is both culturally resonant and visually distinctive.
